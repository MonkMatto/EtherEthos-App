Decentralized Profiles

Embodying Ethereum's Spirit

EtherEthos is a free, decentralized, fully composable profile primitive. It allows any Ethereum address to store and manage its own data, and it allows both smart-contracts and web apps to incorporate this data into their applications. There is no API and no dependencies besides Ethereum. EtherEthos supports account delegation via delegate.xyz and via smart contract ownership.


**Note: user-submitted data is not verified for accuracy or honesty. All user-submitted data is formatted like this, and may include links - please be vigilant. EtherEthos cannot guarantee your safety. Please see all disclaimers of use on the contract.
